# Book store application built with Vue

![image](https://github.com/vnhuyen/vue-book-store/assets/86275176/ba81e687-9bea-4b32-b43b-a14a70c673b1)


## Technologies

- Vue 3
- Typescript
- Vuex/Pinia
- Json Server

## Project Setup

```sh
npm install
```

### Start Fake REST API Server

```sh
json-server --watch db.json
```

### Start App
```sh
npm run dev
```
